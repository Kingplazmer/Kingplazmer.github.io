Navigation bar

2nd Game Project

This is the second game that I am working on and it's still in very early development. The game is planned to be an open world single player adventure game. Some features however have had some progress made on them.

This will be the player's stats during the game. They include hunger, thirst, health and oxygen. The oxygen bar only appears when the player is in water.
These stats will go down over time or when conditions are met like using an item. Thirst and hunger will go down gradually over time and oxygen will go down fast under water. All stats have a limit that it can't go past and will reset to the maximum value if it goes past that limit. The bar will tween and move according to what the status value is to give the player a visual idea of their stats.
All of these stats are saved between game sessions so when a player leaves and then joins back their stats will have saved.
This is the player's inventory in the game. The inventory can easily be opened and closed using the e button. Right now the player is only holding four donuts. The inventory displays the name of the item, an image of the item and how many items are in the players inventory. A player can easily drop an item by then pressing the image of the item.
Once an item is picked up then the code checks and finds the first empty slot or a slot with the same item. Then the item's name, image and number in the inventory is added into the inventory. If the player drops an item then the code checks the number of items after the item has been dropped. If there is still some items left the the number is just reduced by one. Otherwise the slot is reset back to its blank form.
