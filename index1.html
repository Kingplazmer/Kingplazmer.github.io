Library Website project

This is a small project I worked on where the simple goal was to set up a website that connected to a database so that the user could 'register' for a library newsletter. This project used Html, Css and JS. This website was connected to a PHPmyadmin database where the information would be stored.

This is the main page for the website which shows the purpose of the website and acts as a page to direct the user towards the registration page.
The JS code here makes a counter to show the number of times visited (number of times website was loaded).
This page is the registration page where users can put in their details such as their ID, name, Email, suburb and postcode. If the information has been entered correctly and the email is valid then the information then gets sent to the PHPmyadmin database and the user is sent back to the homepage. If the information is not valid the user will receive a notification and be told to re-enter the information.
The PHP code here is used to set up variables for the data being sent to the databse and then send that data once it has all been entered and checked to see if it's valid.
This code is simple html to set up a UI were the user can enter information in text boxes and drop down boxes.
